### **核心武器：`boxplot` (箱线图)**
箱线图是数据分析师的“显微镜”，它能一眼看穿数据的分布特征。

* **它能告诉你什么？**
    1. **中位数**：数据中间在哪里？（箱子中间的那条横线）
    2. **四分位数**：数据的集中区间在哪里？（箱子的上下边缘，即 $Q1$ 到 $Q3$）
    3. **离群点 (Outliers)**：哪些数据是不正常的“显眼包”？（箱子胡须之外的小圆点）
    4. **偏态**：数据是向上挤还是向下挤？



* **代码示例**：
  ```python
  import seaborn as sns
  import matplotlib.pyplot as plt

  # 假设 df 是房价数据，横轴是房间数，纵轴是价格
  sns.boxplot(x='rooms', y='price', data=df)
  plt.show()
  ```

### **为什么有了 Matplotlib 还要用 Seaborn？**

1. **自动美化**：Seaborn 默认的配色方案（如 `deep`, `muted`, `pastel`）比 Matplotlib 的原始色彩要高级得多，直接就能放进 PPT 或论文。
2. **深度集成 Pandas**：它能直接识别 `DataFrame` 的列名作为坐标轴标签，不需要你手动提取 `df['column'].values`。
3. **统计内置**：比如画柱状图时，它会自动帮你计算并画出**置信区间（误差棒）**，这在科研中非常重要。

---

### **Seaborn 的其他高频方法（补充推荐）**

除了箱线图，这几个方法也是必会的：

* **`sns.heatmap()` (热力图)**：
  常配合 `df.corr()` 使用。通过颜色的深浅直接看出哪些变量之间相关性最强（比如“面积”和“总价”通常是深红色）。
  

* **`sns.countplot()` (计数图)**：
  专门统计分类变量的数量。比如：这组数据里有多少男的，多少女的？不需要自己写 `value_counts()`，一句话出图。

* **`sns.pairplot()` (变量关系组图)**：
  **神技！** 一行代码画出数据集中所有数值变量两两之间的散点图和分布图，是做特征工程前的首选探索工具。

### 💡 讲师总结：可视化库的“排兵布阵”
在实际工作中，我们的策略通常是：
* **简单的快速预览**：用 `df.plot()` (Pandas 自带)。
* **统计分析与汇报**：用 `sns.boxplot()` 或 `sns.heatmap()` (Seaborn)。
* **精细化排版/改坐标轴细节**：回到底层用 `plt.xticks()` 或 `plt.legend()` (Matplotlib)。
