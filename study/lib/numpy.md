欢迎来到 Python 进阶的“核心地带”！如果说 Pandas 是处理表格的“办公软件”，那么 **NumPy** 就是支撑所有 AI 和科学计算的“底层引擎”。

在 NumPy 中，一切皆为 **NDArray**（多维数组）。让我们深入拆解你列出的这些高频武器：

---

### 1. 核心容器：`array`
NumPy 的数组与 Python 原生列表最大的区别在于**内存连续性**。
* **特性**：它要求所有元素类型一致（通常是 float 或 int），这使得它可以利用 CPU 的 SIMD 指令集进行并行计算，速度比原生列表快 10 到 100 倍。
* **维度概念**：从标量（0D）到向量（1D），再到矩阵（2D）和张量（ND）。



---

### 2. 逻辑分水岭：`where(condition, x, y)`
这是处理数据时最常用的“开关”。它的逻辑非常像 Excel 里的 `IF` 函数，但它是**矢量化执行**的。
* **语法**：$result = np.where(condition, x, y)$
* **实战演练**：
  ```python
  scores = np.array([55, 80, 45, 95])
  # 成绩 >= 60 记为 "及格"，否则 "不及格"
  status = np.where(scores >= 60, "及格", "不及格")
  ```
* **进阶技巧**：如果只传 `condition`，它会返回满足条件的元素**索引**。

---

### 3. 结果解析器：`argmax`
在深度学习的**多分类任务**中，模型最后输出的通常是一个概率分布（如：`[0.1, 0.8, 0.1]`）。
* **作用**：它不关心最大值是多少，只关心最大值在**哪个位置**。
* **示例**：如果索引 `1` 代表“猫”，那么 `np.argmax([0.1, 0.8, 0.1])` 返回 `1`，我们就知道模型预测结果是猫。

---

### 4. 空间改造师：`expand_dims` 与 `transpose`
这两个方法在处理**图像数据**和**深度学习模型输入**时是必修课。

* **`expand_dims` (维度扩充)**：
  模型通常要求输入是“批处理”模式。如果你只有一张图 $(224, 224, 3)$，模型却要 $(Batch, H, W, C)$。
  `np.expand_dims(img, axis=0)` 就能瞬间把它变成 $(1, 224, 224, 3)$。

* **`transpose` (维度转置)**：
  不同框架对图像的偏好不同：
  * **TensorFlow/OpenCV**: 使用 $[Height, Width, Channel]$
  * **PyTorch**: 使用 $[Channel, Height, Width]$
  使用 `np.transpose(img, (2, 0, 1))` 就可以精准完成这种通道挪位。



---

### 5. 边界定义：`inf` (Infinity)
`np.inf` 代表数学上的 $\infty$。
* **配合 Pandas `cut` 使用**：当你不知道数据的最大上限时，可以用 `inf`。
  ```python
  # 划分等级：0-60, 60-90, 90-无穷大
  bins = [0, 60, 90, np.inf]
  ```
* **注意**：在计算倒数时，如果除以 0，NumPy 通常会返回 `inf` 而不是报错。
