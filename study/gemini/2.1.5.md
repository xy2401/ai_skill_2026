# 2.1.5 健康与营养咨询数据预处理与数据规范设计

## 1. 任务背景与报错分析
本任务涉及健康咨询客户数据的预处理。针对用户提到的 `TypeError: 'value' must be an instance of str or bytes, not a float` 报错，这里进行简要分析：

### 报错原因
在使用 `matplotlib` 绘图（如 `plt.scatter` 或 `plt.pie`）时，如果传递给 **标签（labels）**、**标题（title）** 或 **图例（legend）** 的内容包含 `NaN`（浮点型空值）或者直接传递了数字而非字符串，绘图后端在尝试渲染文字时会抛出此异常。
- **解决方法**：确保绘图相关的标签列已剔除空值（`dropna`），或者使用 `.astype(str)` 显式转换为字符串。

---

## 2. 代码解析

### 2.1 数据加载与初步检查
```python
import pandas as pd

# 加载数据集
data = pd.read_csv('health_consultation.csv')

# 查看表结构基本信息
print(data.info())

# 显示每一列的空缺值数量
print(data.isnull().sum())
```

### 2.2 缺失值与数据类型转换
对年龄列进行强制转换，并剔除无法转换（变成 `NaN`）以及逻辑错误的数据。
```python
# 删除含有缺失值的行
data_cleaned = data.dropna()

# 转换 'Your age' 为整数，errors='coerce' 会将无法转换的值设为 NaN
data_cleaned.loc[:, 'Your age'] = pd.to_numeric(data_cleaned['Your age'], errors='coerce')
# 剔除空值并保留合理年龄
data_cleaned = data_cleaned.dropna(subset=['Your age'])
data_cleaned = data_cleaned[data_cleaned['Your age'] >= 0]
data_cleaned.loc[:, 'Your age'] = data_cleaned['Your age'].astype(int)
```

### 2.3 数据去重
```python
# 检查和删除重复值
duplicates_removed = data_cleaned.duplicated().sum()
data_cleaned = data_cleaned.drop_duplicates()

print(f"Removed {duplicates_removed} duplicate rows")
```

### 2.4 分类特征编码（标签编码）
将文本形式的健身水平转换为数值，以便机器学习模型处理。
```python
from sklearn.preprocessing import LabelEncoder

# 归一化/编码健身水平列
label_encoder = LabelEncoder()
target_col = 'How do you describe your current level of fitness ?'
data_cleaned[target_col] = label_encoder.fit_transform(data_cleaned[target_col])
```

### 2.5 饼图可视化
展示健身频率的分布。注意在绘图前必须处理该列的空值，否则可能触发前述的 `TypeError`。
```python
import matplotlib.pyplot as plt

# 统计频率
exercise_frequency_counts = data_cleaned['How often do you exercise?'].value_counts()

# 绘制饼图
plt.figure(figsize=(10, 6))
exercise_frequency_counts.plot.pie(autopct='%1.1f%%', startangle=90, colors=plt.cm.Paired.colors)
plt.title('Distribution of Exercise Frequency')
plt.ylabel('') # 隐藏默认的纵向标签
plt.show()
```

### 2.6 数据划分与保存

from sklearn.model_selection import train_test_split

## 3. 数据清洗与标注规范建议
1. **数据一致性规范**：
   - 文本列统一去除首尾空格（`str.strip()`）。
   - 统一大小写（如 `Yes/No` 统一为首字母大写）。
2. **异常值处理规范**：
   - 年龄、身高、体重应设置合理的业务阈值。
   - 对于极端异常值，需明确是剔除还是修正。
3. **标注规范**：
   - 对于分类特征，需维护一份 `LabelEncoder` 的映射字典（Mapping），确保训练集和推理集编码一致。
   - 标注过程中若存在主观判断（如“健身水平”），应提供详细的分级描述参考手册。
