# 1.1.2 智能农业系统中的业务数据采集和处理流程设计 - 题目解析

这道题目要求处理农田传感器数据，完成基础统计、位置维度分析以及数据清洗。

---

### 1. 传感器数据统计 (Task 1)

任务是按传感器类型统计数据的条数（count）和平均值（mean）。

```python
# 读取数据集
data = pd.read_csv('sensor_data.csv')

# 对传感器类型进行分组，并计算 'Value' 列的 count 和 mean
# agg() 方法允许我们在分组上同时运行多个聚合函数
sensor_stats = data.groupby('SensorType')['Value'].agg(['count', 'mean'])
```

### 2. 按位置统计温湿度 (Task 2)

需要筛选出温度和湿度数据，并按安装位置查看它们的平均表现。

```python
# 使用 isin() 过滤特定类型的传感器
# unstack() 用于将分组产生的多级索引转换为透视表格式，使 Location 为行，SensorType 为列
location_stats = data[data['SensorType'].isin(['Temperature', 'Humidity'])].groupby(['Location', 'SensorType'])['Value'].mean().unstack()
```

### 3. 数据清洗与异常值处理 (Task 3)

根据业务逻辑（温度和湿度的合理范围）标记异常值，并处理缺失数据。

```python
# 标记异常值：np.where(条件, 真值, 假值)
# 逻辑：(是温度且不在 -10~50 之间) 或 (是湿度且不在 0~100 之间)
data['is_abnormal'] = np.where(
    ((data['SensorType'] == 'Temperature') & ((data['Value'] < -10) | (data['Value'] > 50))) |
    ((data['SensorType'] == 'Humidity') & ((data['Value'] < 0) | (data['Value'] > 100))),
    True, False
)

# 统计异常值数量
print("异常值数量:", data['is_abnormal'].sum())

# 填补缺失值：优先用前一个有效值 (ffill)，若还缺失则用后一个有效值 (bfill)
# 注意：新版 Pandas 推荐直接使用 .ffill() 和 .bfill()，而不是 fillna(method='ffill')
data['Value'].ffill(inplace=True)
data['Value'].bfill(inplace=True)

# 保存清洗后的数据
# 使用 drop 移除辅助用的 'is_abnormal' 列
cleaned_data = data.drop(columns=['is_abnormal'])
cleaned_data.to_csv('cleaned_sensor_data.csv', index=False)
```

---

### 关键点总结：
1.  **分组聚合**: `groupby` 配合 `agg` 是处理多种统计指标的利器。
2.  **透视转换**: `unstack()` 能够让统计结果更加直观。
3.  **数据填补**: `ffill` (forward fill) 和 `bfill` (backward fill) 是处理传感器连续数据的标准手段。
4.  **逻辑过滤**: `isin` 和 `np.where` 用于处理复杂的筛选和分类逻辑。
