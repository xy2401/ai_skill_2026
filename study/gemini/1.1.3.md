### 1. 数据完整性审核 (Task 1)

检查数据是否有缺失（NaN）或录入重复。

```python
# isnull().sum() 统计每列缺失值
missing_values = data.isnull().sum()

# duplicated().sum() 统计重复行数
```

### 2. 数据合理性审核 (Task 2)

根据金融业务规则设定数据边界。

```python
# 年龄审核：18-70岁
data['is_age_valid'] = data['Age'].between(18, 70)

# 收入审核：大于 2000
data['is_income_valid'] = data['Income'] > 2000

# 贷款额度审核：应小于收入的 5 倍
data['is_loan_amount_valid'] = data['LoanAmount'] < (data['Income'] * 5)

# 信用评分审核：300-850 分
data['is_credit_score_valid'] = data['CreditScore'].between(300, 850)
```

### 关键点总结：
1.  **`between` 函数**: 判定数值范围的最简便方法。
2.  **逻辑组合**: 利用 `all(axis=1)` 可以快速判断一行数据是否“全字段达标”。
