# 1.1.1 智能医疗系统中的业务数据处理流程设计 - 题目解析

这道题目要求使用 Python（主要是 `pandas` 和 `numpy` 库）对一份医疗数据进行预处理和统计分析。主要任务是根据住院天数定义“高风险”和“低风险”患者，并观察风险等级在不同 BMI（体重指数）区间和年龄区间下的分布情况。

---

### 1. 数据读取与基础统计 (Task 1)

首先需要读取数据，并根据住院天数（`DaysInHospital > 7`）创建一个标识风险等级的新列。

```python
# 读取数据集
data = pd.read_csv('patient_data.csv')

# (1) 统计住院天数超过7天的患者数量及其占比
# np.where(条件, 符合条件的值, 不符合条件的值) 是常用的快速分类方法
data['RiskLevel'] = np.where(data['DaysInHospital'] > 7, '高风险患者', '低风险患者')

# 统计不同风险等级的数量，使用 value_counts()
risk_counts = data['RiskLevel'].value_counts()

# 计算占比，分母是数据的总条数 len(data) 或 risk_counts.sum()
high_risk_ratio = risk_counts['高风险患者'] / len(data)
low_risk_ratio = risk_counts['低风险患者'] / len(data)
```

### 2. BMI 区间分析 (Task 2)

这一步考察如何使用 `pd.cut` 将连续的数值（BMI）离散化到指定的区间中，并进行分组统计。

*   **区间定义**：题目要求 `偏瘦(<18.5)`、`正常(18.5-23.9)`、`超重(24-27.9)`、`肥胖(>=28)`。
*   **代码实现**：注意 `right=False` 表示左闭右开区间 `[left, right)`。

```python
# (2) 统计不同 BMI 区间中高风险患者的比例
# pd.cut 会根据 bins 边界将数据贴上 labels 标签
data['BMIRange'] = pd.cut(data['BMI'], bins=bmi_bins, labels=bmi_labels, right=False)

# 使用 groupby 分组，计算每个区间内高风险患者出现的平均频率（即比例）
bmi_risk_rate = data.groupby('BMIRange')['RiskLevel'].apply(lambda x: (x == '高风险患者').mean())

# 统计每个区间的总人数
bmi_patient_count = data['BMIRange'].value_counts()
```

### 3. 年龄区间分析 (Task 3)

逻辑与 BMI 分析完全一致，只是分箱的边界（`age_bins`）和标签（`age_labels`）不同。

```python
# (3) 统计不同年龄区间中高风险患者的比例
data['AgeRange'] = pd.cut(data['Age'], bins=age_bins, labels=age_labels, right=False)

# 同样使用 groupby 计算风险比例
age_risk_rate = data.groupby('AgeRange')['RiskLevel'].apply(lambda x: (x == '高风险患者').mean())

# 统计每个年龄区间的总人数
age_patient_count = data['AgeRange'].value_counts()
```

---

### 关键点总结：
1.  **`np.where`**: 用于基于单个条件进行二分类。
2.  **`pd.cut`**: 重点在于理解 `bins` 边界和 `right=False` 的配合，确保符合题目要求的“低于”、“～”、“及以上”的逻辑。
3.  **`groupby` + `apply`**: 这是处理“分组占比”问题的标准流程。`mean()` 在布尔值（True/False）上运行会直接得到 True 的比例。
4.  **`value_counts()`**: 快速统计每个类别的频数。
