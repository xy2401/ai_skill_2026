## 1. Python 代码解析

以下是完成任务所需的 Python 代码及其详细解析：

```python
import pandas as pd
from sklearn.preprocessing import StandardScaler
from sklearn.model_selection import train_test_split

# (1) 正确加载数据集并显示前五行及数据类型
# 使用 pandas 的 read_csv 函数读取本地 CSV 文件
data = pd.read_csv('auto-mpg.csv')
print("数据集的前五行:")
print(data.head()) # 显示前五行
print("\n数据类型:")
print(data.dtypes) # 显示每一列的数据类型

# (2) 检查缺失值并删除缺失值所在的行
# isnull().sum() 用于统计每列的缺失值数量
print("\n检查缺失值:")
print(data.isnull().sum())
# dropna() 函数默认删除所有包含缺失值的行
data = data.dropna()

# (3) 将 “horsepower” 列转换为数值类型，并处理异常值
# horsepower 列中可能含有 '?' 等非数值字符。
# errors='coerce' 会将无法转换的值设为 NaN
data['horsepower'] = pd.to_numeric(data['horsepower'], errors='coerce')
# 转换后再次删除由于转换产生的缺失值
data = data.dropna(subset=['horsepower'])
print("\n清洗后的 horsepower 类型:", data.horsepower.dtypes)

# (4) 对数值型数据进行标准化处理
# 标准化（Standardization）使数据符合标准正态分布（均值为0，标准差为1）
numerical_features = ['displacement', 'horsepower', 'weight', 'acceleration']
scaler = StandardScaler()
# 使用 fit_transform 进行拟合及转换
data[numerical_features] = scaler.fit_transform(data[numerical_features])

# (5-7) 特征选择、目标变量标注及数据划分
# 根据业务需求选择对燃油效率（mpg）预测有用的特征
selected_features = ['cylinders', 'displacement', 'horsepower', 'weight', 'acceleration', 'model year', 'origin']
X = data[selected_features] # 自变量
y = data['mpg']             # 目标变量（已标注）

# 划分数据集：训练集占 80%，测试集占 20%
X_train, X_test, y_train, y_test = train_test_split(X, y, train_size=0.8, random_state=42)

# (8) 保存处理后的数据
# 将特征和目标变量合并后保存
cleaned_data = X.copy()
cleaned_data['mpg'] = y
# index=False 表示不保存行索引
cleaned_data.to_csv('2.1.1_cleaned_data.csv', index=False)
print("\n清洗后的数据已保存到 2.1.1_cleaned_data.csv")
```

## 2. 数据清洗和标注流程规范设计

为了确保模型训练的质量，需制定以下规范：

### 数据清洗规范
1.  **缺失值处理**：
    *   对于数值型特征，如果缺失比例较低（如 <5%），直接删除含缺失值的行。
    *   如果缺失比例较高，考虑使用中位数或均值进行填充，或者利用相关性进行插值。
2.  **类型转换与异常处理**：
    *   确保所有特征列（如 `horsepower`）的数据类型与其实际物理意义相符（应为 float/int）。
    *   对于非数值字符（如 `?`），应转换为 `NaN` 后按照缺失值逻辑处理。
3.  **标准化/归一化**：
    *   对于量纲差异较大的特征（如 `weight` 和 `acceleration`），必须进行标准化处理，防止模型被大数值特征主导。

### 数据标注规范
1.  **明确目标变量**：
    *   将汽车燃油效率（`mpg`）明确标注为回归任务的目标变量。
2.  **特征关联性选择**：
    *   选择与 `mpg` 强相关的物理特征（气缸数、排量、马力、车重等）作为模型输入。
    *   移除与物理预测无关的描述性特征（如 `car name`）。
3.  **数据集划分一致性**：
    *   采用固定的随机种子（如 `random_state=42`）进行划分，确保实验结果的可重复性。
    *   训练集与测试集应保持相同的分布。
